CheckSudokuIsSolved is a python program that accepts a finished 2D matrix sudoku puzzle as it's input and decides on whether or not the puzzle is a valid solution by checking the row, column, and square rules.

MatrixMult is a python program that requests the number of rows and columns of two different matrixes and computes the matrix that is the result of multiplying the two matrixes together.

RomanNum is a python program that's input is a base ten integer. The program then converts the given number to its roman numeral equivalent.

RoamnNumDecoder is a python program that is given a roman numeral and converts it to its base ten equivalent. 

StripGivenMarkers is a python program that strips all text that follows any of a set of comment markers passed in. Any whitespace at the end of the line should also be stripped out.

Ex. "apples, pears # and bananas\ngrapes\nbananas !apples" should become "apples, pears\ngrapes\nbananas"

CountHits is a python program that finds all possible contiguous perfect squares of specific power sequences. (still working on a polynomial time variant). There are comments at the bottom of the file that describe what is going on.\\

LongestSubstring is a python program that implements the backtracking algorithm to find the longest common substring between two strings.
